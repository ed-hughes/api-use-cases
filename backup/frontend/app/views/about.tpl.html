Jotform Back-up Service is a small web-app that you can use to safely back-up your jotform data. Currently we back-up your data into our servers in JSON format. But this service is under development and many more usefull features will be released. You can check release log below on this page.

Currently Implemented Features

  • Back-Up of user form data.
  • Back-Up of user submission data.

Upcoming features of Jotform Back-Up service

  • Complete Form&Submissions viewer for backed-up data.
  • Back-Up uploaded files.
  • Allow users to download their back-ups to their local computer.
  • Re-Load already backed-up data into user's Jotform account or a different user's Jotform account allowing migration of data between accounts.

Contribute, report bugs or feature suggestions

This project is part of our api-use-cases/backup github report which includes many sub projects to demonstrate basic usage of jotform api. Feel free to send pull requests.
